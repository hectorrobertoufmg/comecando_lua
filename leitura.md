# For Loop
O for loop em lua é mais simples de escrever, por exemplo, suponhamos que exista um array:

copas = {1958, 1962, 1970, 1994, 2002}

para correr o array, usamos
~~~
for i=1, #copas do
    print(copas[i])
end
~~~
